================================================================================
PROBLEM STATEMENT
================================================================================

Business Challenge: Small and medium businesses struggle with inventory management
and pricing decisions due to:
- Lack of data-driven decision making
- Manual competitor price tracking
- Inaccurate demand forecasting
- Poor understanding of market factors
- High risk of stockouts or overstocking
- Limited access to business intelligence tools

Impact: Leads to lost revenue, wasted inventory, and poor customer satisfaction.

================================================================================
SOLUTION OVERVIEW
================================================================================

A comprehensive AI-powered business intelligence platform that provides:
- Smart Demand Prediction using machine learning
- Real-time Competitor Price Tracking via web scraping
- Multi-factor Analysis (stock levels, sales velocity, market conditions)
- Automated Decision Recommendations with confidence scoring
- Interactive Dashboards for data visualization
- Historical Data Tracking for continuous improvement

================================================================================
SOLUTION WORKFLOW
================================================================================

1. Data Input Phase:
   User Input → Product Details → Market Analysis → AI Processing → Decision Output

   Inputs Required:
   - Product Name
   - Current Stock Quantity
   - Recent Sales Count (last period)
   - Current Product Price

2. Data Processing Phase:
   CSV Historical Data + Web Scraping + ML Algorithm → Demand Prediction

   Data Sources:
   - Historical product data (CSV)
   - Live competitor prices (Amazon, Flipkart, Meesho)
   - Market factors and trends

3. AI/ML Processing:
   Linear Regression + Rule-based Logic → Confidence Score + Recommendation

   Algorithm: Linear regression combined with business rules for demand forecasting.

4. Output Generation:
   Prediction Results → Visual Dashboard → Actionable Insights

TECHNOLOGY STACK
================================================================================

FRONTEND (React/TypeScript):
- Framework: React 18 with TypeScript
- Build Tool: Vite (lightning-fast development)
- Styling: Tailwind CSS (utility-first)
- Charts: Recharts (interactive data visualization)
- State Management: React Hooks
- HTTP Client: Fetch API with CORS support

BACKEND (Java/Servlet):
- Runtime: Java 17
- Web Framework: Jakarta Servlet 5.0
- Application Server: Jetty 11
- Build Tool: Maven
- ML Library: Apache Commons Math 3.6.1
- Web Scraping: Selenium WebDriver + WebDriverManager

DATA & STORAGE:
- Data Format: CSV files
- ML Data: Historical product performance data
- Configuration: Properties files

DEPLOYMENT & HOSTING:
- Frontend: Netlify (free tier) or Azure Static Web Apps
- Backend: Azure Container Apps (consumption plan - pay per use)
- CI/CD: GitHub Actions
- Infrastructure: Azure Bicep templates

DEVELOPMENT TOOLS:
- IDE: VS Code
- Version Control: Git + GitHub
- Package Management: npm (frontend), Maven (backend)
- Testing: Manual testing + build verification

QUICK START GUIDE

   cd Muti-factor-business-system

2. START BACKEND:
   mvn clean package
   mvn jetty:run

3. START FRONTEND (New Terminal):
   cd project
   npm install
   npm run dev

4. ACCESS APPLICATION:
   Frontend: http://localhost:5173
   Backend API: http://localhost:8090/decision
