# Multi-Factor Business System - Requirements & Dependencies

## SYSTEM REQUIREMENTS

### Minimum Hardware
- RAM: 4GB
- Storage: 500MB free space
- OS: Windows 10+, macOS 10.15+, Linux (Ubuntu 18.04+)

### Software Prerequisites
- Java Development Kit (JDK) 17 or higher
- Node.js 16 or higher
- Apache Maven 3.6 or higher
- Git 2.0 or higher

VITE_BACKEND_URL=https://your-backend-url-here.azurecontainerapps.io

## PORT CONFIGURATION

### Development
- Frontend: http://localhost:5173
- Backend: http://localhost:8090

### Production
- Frontend: https://your-netlify-site.netlify.app
- Backend: https://your-app-name.region.azurecontainerapps.io

## DATA REQUIREMENTS

### CSV Data Format (data/Products.csv)
Columns: ProductName,Stock,Sales,Demand,Price,Category
Format: CSV with headers
Example:
ProductName,Stock,Sales,Demand,Price,Category
Wireless Headphones,50,40,75,2500,Electronics
Laptop,25,20,45,45000,Electronics

## BUILD COMMANDS

### Frontend
npm install          # Install dependencies
npm run dev         # Start development server
npm run build       # Build for production
npm run preview     # Preview production build

### Backend
mvn clean           # Clean build artifacts
mvn compile         # Compile Java sources
mvn package         # Create WAR file
mvn jetty:run       # Start development server

## DEPLOYMENT REQUIREMENTS

### Netlify (Frontend)
- Build command: npm run build
- Publish directory: project/dist
- Environment variables: VITE_BACKEND_URL

### Azure Container Apps (Backend)
- Docker image required
- Consumption plan (pay-per-use)
- Minimum CPU: 0.25 vCPU
- Minimum RAM: 0.5 GB

## NETWORK REQUIREMENTS

### Development
- Localhost access on ports 5173, 8090
- Internet access for web scraping

### Production
- HTTPS required
- CORS configuration
- Firewall rules for Azure services
